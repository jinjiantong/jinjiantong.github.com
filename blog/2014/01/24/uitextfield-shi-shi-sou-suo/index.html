UITextField 实时搜索

Jan 24th, 2014

项目的需求 UITextField 根据用户输入实时搜索，当用户连续输入的时候就不查询数据库，这些根据用户输入的时间间隔来判断。

[_keyTF addTarget:self action:@selector(searchBeginClick:) forControlEvents:UIControlEventEditingChanged];

  • (void) searchBeginClick:(id)sender {

    [self.class cancelPreviousPerformRequestsWithTarget:self selector:@selector(updateData) object:self];

    [self performSelector:@selector(updateData) withObject:self afterDelay:0.5];

}

–(void)updateData { //查询数据库 }

在这里时间间隔为0.5秒，也可以作为参数提取出来自定义时间间隔。
